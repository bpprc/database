Get the development version of BBPRC database from Github


If you have Python, Git and pip installed, use the following commands:


git clone https://github.com/Amrithasuresh/BPPRC.git

cd bpprc

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

Then copy the following URL in your browser

http://127.0.0.1:8000/
